# Meta-analysis

## Fixed effect IVW

$$ \bar{\beta} = {{\sum_{i=1}^{k} {w_i \beta_i}}\over{\sum_{i=1}^{k} {w_i}}} $$

- $w_i = 1 / \delta^2_i$


## 

## Heterogeneity test

!!! info "Cochran's Q test"

    $$ Q = \sum_{i=1}^{k} {w_i (\beta_i - \bar{\beta})^2} $$
    
    

!!! info "$I^2$"
    $$ I^2 =  {{Q - df}\over{Q}}\times 100% =  {{Q - (k - 1)}\over{Q}}\times 100% $$
